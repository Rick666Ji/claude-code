# Claude Code - 桌面应用\n\n一个现代化的 AI 驱动代码编辑器，采用 Electron 和 React 构建。这是 Claude Code 的入口界面，具备项目管理、模板和设置功能。\n\n## 🚀 快速开始\n\n### 方式 1：使用批处理文件（推荐）\n\n1. **克隆项目到 D 盘**\n```bash\ngit clone https://github.com/Rick666Ji/claude-code.git D:\\claude-code\ncd D:\\claude-code\nnpm install\n```\n\n2. **双击文件启动应用**\n   - 直接双击 `启动Claude Code.bat` 文件\n   - 应用会自动启动\n\n### 方式 2：使用 VBS 脚本（无命令行窗口）\n\n双击 `start-claude-code.vbs` 文件\n- 应用在后台启动\n- 不会显示命令行窗口\n- 更加整洁\n\n### 方式 3：命令行启动\n\n```bash\ncd D:\\claude-code\nnpm run dev\n```\n\n---\n\n## 📋 功能特性\n\n✨ **主页界面**\n- 现代深色主题 UI，渐变背景\n- 快速项目操作（创建/打开）\n- 最近项目列表管理\n- 项目模板快速启动\n- 文档和支持链接\n- 可自定义的设置面板\n\n🎨 **用户友好的组件**\n- 带模板选择的创建项目对话框\n- 文件浏览器打开项目\n- 主题、外观和行为设置\n- 关于应用的信息对话框\n- 平滑的动画和过渡\n\n🚀 **快速启动**\n- 支持模板（React、Next.js、Full Stack、Vanilla JS）\n- 项目创建向导\n- 最近项目追踪\n- 一键打开项目\n\n---\n\n## 📦 项目结构\n\n```\nclaude-code/\n├── src/\n│   ├── main/\n│   │   ├── index.js          # Electron 主进程\n│   │   └── preload.js        # IPC 通道\n│   ├── pages/\n│   │   └── HomePage.js       # 主页\n│   ├── components/\n│   │   ├── NewProjectDialog.js\n│   │   ├── CreateProjectDialog.js\n│   │   ├── SettingsPanel.js\n│   │   └── AboutModal.js\n│   ├── styles/\n│   │   ├── HomePage.css\n│   │   ├── Dialog.css\n│   │   ├── CreateProjectDialog.css\n│   │   ├── SettingsPanel.css\n│   │   └── App.css\n│   ├── App.js\n│   ├── App.css\n│   └── index.js\n├── public/\n│   └── index.html\n├── 启动Claude Code.bat       # Windows 批处理启动文件\n├── start-claude-code.vbs     # VBS 无窗口启动脚本\n├── package.json\n└── README_CN.md\n```\n\n---\n\n## 🔧 安装依赖\n\n首次使用需要安装依赖：\n\n```bash\ncd D:\\claude-code\nnpm install\n```\n\n---\n\n## 🎮 使用方式\n\n### 方式 1：双击批处理文件\n**最简单的方式 - 适合所有人**\n\n- 在 `D:\\claude-code` 文件夹中找到 `启动Claude Code.bat`\n- 双击文件\n- 应用启动，会看到命令行窗口显示进度\n\n### 方式 2：双击 VBS 脚本\n**最干净的方式 - 没有命令行窗口**\n\n- 在 `D:\\claude-code` 文件夹中找到 `start-claude-code.vbs`\n- 双击文件\n- 应用在后台启动\n- 看到一个启动提示框\n\n### 方式 3：创建桌面快捷方式\n**最方便的方式 - 放在桌面**\n\n1. 右键点击 `启动Claude Code.bat`\n2. 选择 **发送到** → **桌面（快捷方式）**\n3. 以后直接在桌面双击快捷方式启动\n\n---\n\n## 🎨 主题和设置\n\n应用启动后，你可以在设置中自定义：\n\n- **主题**：亮色/深色\n- **字体大小**：12px - 18px\n- **行高**：1.0 - 2.0\n- **自动更新**：开/关\n- **遥测数据**：开/关\n\n---\n\n## ⌨️ 快捷键\n\n- `Ctrl + N` - 新建项目\n- `Ctrl + O` - 打开项目\n- `Ctrl + Q` - 退出应用\n\n---\n\n## 💻 系统要求\n\n- Windows 7 或更高版本\n- Node.js 14+\n- npm 6+\n\n---\n\n## 📚 使用的技术\n\n- **Electron** - 桌面应用框架\n- **React 18** - UI 库\n- **React Icons** - 图标库\n- **Electron Builder** - 应用打包\n\n---\n\n## 📖 文档和支持\n\n- 📖 [官方文档](https://docs.claudecode.dev)\n- 🐛 [报告问题](https://github.com/anthropic/claude-code/issues)\n- 💬 [社区论坛](https://claudecode.dev/community)\n- 📧 [联系支持](https://claudecode.dev/support)\n\n---\n\n## 📄 许可证\n\nMIT License -
